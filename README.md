# 💊 MedFlow - Smart Medicine Reminder System

## 📌 Overview

MedFlow is a full-stack medicine reminder and health management application that helps users manage medications, track schedules, and receive reminders.

## 🚀 Features

* User Registration & Login
* Add, Edit, and Delete Medicines
* Medicine Reminder Scheduling
* Dashboard with Medication Tracking
* Statistics and Reports
* Dark/Light Theme
* Responsive Design
* MongoDB Atlas Integration
* Cloud Deployment

## 🛠 Tech Stack

### Frontend

* React.js
* Vite
* JavaScript
* CSS

### Backend

* Node.js
* Express.js

### Database

* MongoDB Atlas
* Mongoose

### Deployment

* Render
* GitHub
